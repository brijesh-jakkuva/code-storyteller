# Change Log

## 0.1.0

Initial release.

### Features
- Tell Story for file — narrate any supported source file
- Tell Story for selection — narrate highlighted code
- Diff Story (Git) — narrate changes between commits
- Project Story — narrate entire folder
- Show History — view past stories

### Supported Languages
Python, JavaScript, TypeScript, Go, Rust, Java

### Story Styles
heist, recipe, 5yo, pm, sports, detective, fairy_tale
